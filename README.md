# Simple Tkinter App
This is a beginner project to demonstrate how to create a<br>simple GUI application using Python's Tkinter library.

## Requirements
- Python 3.x
- Tkinter

## Setup
```bash
    # virtualenv
    make virtual
    source venv/bin/activate (Linux)
    venv\Scripts/activate    (Windows)

    # install dependencies
    make install

    # run project
    make run
```
